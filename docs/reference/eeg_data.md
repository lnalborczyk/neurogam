# Simulated EEG data

Simulated EEG data.

## Usage

``` r
eeg_data
```

## Format

A data frame with 502000 rows and 5 variables:

- participant:

  Character variable indicating the participant's ID.

- condition:

  Integer variable indicating the condition.

- trial:

  Integer variable indicating the trial number.

- time:

  Numeric variable indicating the time step (in seconds).

- eeg:

  Numeric variable indicating the EEG signal.
